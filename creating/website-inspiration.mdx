## How it works

Paste a website URL into the chat and Fastshot will analyze it to inspire your mobile app's design and content structure.

The system uses a headless browser to visit the website and extract:

- **Visual theme** — colors, fonts, imagery
- **Navigation structure** — menus, page hierarchy
- **Content sections** — headers, text blocks, media
- **Interactive elements** — forms, buttons, CTAs

## Using a website URL

1. Paste the URL directly into the chat input
2. Add context about what you want: "Build a mobile app inspired by this website"
3. Send the message

The analysis takes under 60 seconds. Once complete, the AI uses the extracted design information to generate your app.

<Tip>
  You can combine a website URL with additional instructions. For example: "Use the color scheme from this site but simplify the navigation to 3 tabs."
</Tip>

## What gets analyzed

| Element | What's Extracted |
|---------|-----------------|
| Colors | Primary, secondary, and accent colors from the site's CSS |
| Typography | Font families, sizes, and weights |
| Images | Key visuals and their placement |
| Navigation | Menu structure and page links |
| Layout | Content sections and their arrangement |
| Interactive elements | Forms, buttons, and input fields |

## Tips

- **Public pages only** — the scraper can only access publicly visible pages (no login-required content)
- **Homepage is primary** — the homepage plus a few key linked pages are analyzed
- **Combine with text** — describe what aspects of the website you want to keep or change
- **Works with any site** — e-commerce, blogs, portfolios, SaaS landing pages, etc.
